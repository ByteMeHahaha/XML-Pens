## Important Notes

- This XML project was tested in Firefox and MS Edge.
- This project is intended to be a skills reference only. It serves little to
  no use in a production environment.


## Documents in This Repository

### `Pens.xml`

`Pens.xml` contains the actual XML data for the pens.

### `Pens.xsd`

`Pens.xsd` defines the elements required to create a valid `Pens.xml` document.

### `Pens.xsl`

`Pens.xsl` is the XSL stylesheet for `Pens.xml`.
